# BOJ 15685 - 드래곤 커브

## 알고리즘 태그
- 시뮬레이션(빡구현)

## 문제 링크
https://www.acmicpc.net/problem/15685


## 풀이 아이디어
-미로찾기처럼 하나씩 그려나가는 문제  
-세대가 증가되는 방법  
0:0  
1: 0 1  
2: 0 1 2 1  
3: 0 1 2 1 2 3 2 1  
전 세대의 뒤에서부터 방향(숫자)에서 1를 더해서 추가하면 됨.   
각 드래곤 커브를 칠해주면 끝남.  

### 바킹독의 풀이 아이디어
내 코드랑 비슷함.  
다른 점은 나는 세대를 업데이트하면서 좌표평면에 표시했다면, 바킹독은 미리 드래곤 커브를 완성시키고 표시했음. 그래서 코드가 더 간결해짐.
